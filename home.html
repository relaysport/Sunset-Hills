Sunset Hills

Sunset Hills requires understanding of arrays, array manipulation, and the transferring of information between different coding languages to determine which buildings are lit by sun. To elaborate on the challenge, six buildings are placed in a row with the sun shining against them from the side. The height of any given building, bar the first building, must be greater than the height of those before it for the sun to shine on it. Height values are inputs from the site visitor, my code gives an output of the sunlit buildings. JavaScript, Bootstrap, HTML, and CSS are used in my program to complete the challenge.
